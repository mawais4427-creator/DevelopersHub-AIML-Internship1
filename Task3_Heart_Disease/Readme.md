### Task 3: Heart Disease Prediction
- **Status:** ✅ Completed
- **Objective:** Predict whether a patient has heart disease
- **Dataset:** Heart Disease UCI Dataset - 303 patients, 14 features
- **Libraries:** pandas, numpy, matplotlib, seaborn, scikit-learn
- **Models:** Logistic Regression, Decision Tree Classifier
- **Results:**
  - Logistic Regression Accuracy: ~85%
  - ROC-AUC Score: ~0.91
- **Visualizations:** Age distribution, Correlation heatmap,
  Target distribution, Confusion matrix, ROC curve
- **Key Finding:** Chest pain type and max heart rate are
  the strongest predictors of heart disease
